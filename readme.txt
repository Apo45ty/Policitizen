Installation:
1) Install Apache version 2.2 in your machine.
2) Extract the ".zip" file containing policitizen into you Apache root dirrectory
3) Start a shell terminal and navigate to the apache root dirrectory and run the command:
   "java -jar applicationServer2.jar 60000 g18 18gMAH"
4)Done.
